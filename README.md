# 🍽️ DishCraft

## 📌 Overview
**DishCraft** is a web-based recipe discovery application that enables users to quickly find cooking instructions for their favorite meals. By entering the name of a dish, users can explore complete recipe details including ingredients, preparation steps, and a guided video tutorial.

The application focuses on making cooking simple, interactive, and accessible by presenting all essential information in one place.

---

## 🎯 Objective
This project is built to strengthen core frontend development skills such as working with APIs, managing dynamic data, and implementing user-friendly features like searching, filtering, and sorting.

---

## 🔌 Data Source
This application integrates with the **TheMealDB API** to fetch recipe data dynamically.

### How the API is used:
- Accepts user input (dish name)
- Fetches matching recipes from the database
- Displays:
  - Recipe title
  - Ingredient list with quantities
  - Cooking instructions
  - Category/type of dish
  - Video tutorial link

Using the API removes the need for manual data storage and ensures updated, real-time results.

---

## ✨ Core Features

### 🔎 Smart Search
- Search recipes instantly by typing the dish name  
- Dynamic results based on user input  

### 🧩 Category Filtering
- Narrow down recipes by categories such as:
  - Vegetarian  
  - Dessert  
  - Meat-based dishes  
- Improves user experience by refining results  

### ↕️ Sorting Functionality
- Organize recipes:
  - Alphabetically (A–Z)  
  - Based on ingredient count (complexity)  

### 📖 Detailed Recipe View
- Expand any recipe to view:
  - Complete ingredient list  
  - Step-by-step preparation instructions  
  - Embedded YouTube tutorial for guidance
